🔒 Security & Privacy

Mai Sakurajima is built with security, privacy, and server safety as a priority.
We aim to provide powerful features while keeping your server data protected and your community secure.

🛡️ Data Protection

Mai Sakurajima only stores the data required for bot functionality, such as:

Server configuration settings
Economy and leveling data
Ticket information
Temporary moderation logs
User preferences related to bot systems

Sensitive personal information is never sold or shared with third parties.

🔐 Secure Permissions

The bot only uses permissions necessary for its features to function properly.
Server administrators have full control over:

Command permissions
Staff access
Feature enabling/disabling
Role management systems
Verification settings
👤 User Privacy

Mai Sakurajima respects user privacy.

The bot does not:

Read private direct messages unless required by a command
Collect passwords, tokens, or personal credentials
Track users outside servers where the bot exists
Store unnecessary personal information
🚫 Abuse Prevention

Security systems are included to help protect communities from:

Raids
Spam
Malicious behavior
Unauthorized access
Verification bypass attempts

Moderation and verification tools are designed to improve overall server safety.

📂 Data Storage

Stored data is handled securely and only retained for feature functionality.
Server owners may request data resets or removal where applicable.

⚡ Reliability & Stability

Mai Sakurajima is actively maintained and optimized for:

Stable performance
Fast response times
Reliable uptime
Safe command handling
✅ Transparency

We believe in transparency and responsible bot development.
Our goal is to create a trusted Discord experience focused on:

Community safety
Responsible moderation
Privacy respect
Secure systems
